<h1>NBZ</h1>

Bot to automate browser navigation with self-scripting language named nbz.

<b>Download:</b> https://github.com/zurdi15/NBZ/releases

<b>Documentation:</b> https://github.com/zurdi15/NBZ/wiki

Compatible OS:

  - Windows
    - Chrome
    - Firefox
    - PhantomJS
    
  - Linux
    - Chrome
    - Firefox
    - PhantomJS
    
  - macOS:
    - Chrome

 <h3>Prequisites</h3>
